Langkah 1

Titik awal Anda

Default di atas adalah koordinat Graha Rekso — ganti sesuai titik keberangkatan Anda hari ini, lalu klik "Konfirmasi koordinat".

Langkah 2

Masukkan data merchant

Pilih salah satu: tempel data manual, atau upload file di bawahnya.

Tempel langsung dari Excel/Google Sheets (kolom: Nama Merchant, Alamat Merchant, Latitude, Longitude — dipisah tab atau koma). Baris header opsional, akan terdeteksi otomatis.
